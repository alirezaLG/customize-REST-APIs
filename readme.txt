This plugin create custom Rest API for Mobile or Single page application or even universal app, 


== Description ==

This plugins help you to use Rest API for blogs and news agencies and add some content to the wp-json like menu and custome image and app name. 
this plugin is for integration with Android and iOS applications and universal applications
